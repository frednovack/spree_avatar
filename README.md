# Spree Avatar
Adds an Avatar to Spree User (4.1 ready).

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'spree_avatar', github: 'MateoLa/spree_avatar'
```

And then execute:
```bash
$ bundle update
```

```rails
$ rails g spree_avatar:install
```
