# Spree Avatar
RoR plugin to use with Spree.
Adds an avatar to the user account.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'spree_avatar', github: 'MateoLa/spree_avatar'
```

And then execute:
```bash
$ bundle
```

```rails
$ rails g spree_avatar:install
```
